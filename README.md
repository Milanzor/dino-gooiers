# Dino Gooiers 🦕

A polished Angry Birds-style mobile game — but with dinosaurs and a construction crane!

**Play it:** https://milanzor.github.io/dino-gooiers/

## Over het spel

Gooi goede dino's van een bouwkraan naar de forten van de slechte Tirannen. Vernietig hun bolwerken en red de dino-eieren!

- 🏗️ Bouwkraan launcher (geen katapult!)
- 🦖 6 speelbare helden met unieke krachten
- 👹 5 soorten vijanden (de Tirannen)
- 🗺️ 30 niveaus in 3 werelden
- ⭐ 3-sterren beoordelingssysteem
- 🥚 Ei-broeden & collectie
- 🎩 Hoeden & skins

## Tech Stack

- Vanilla HTML5 Canvas
- Matter.js physics
- No frameworks — browser-native

## Werelden

1. **Jungle Ruïnes** — tropisch, oud steen
2. **Steen Stad** — middeleeuwse stadsmuren
3. **Stalen Burcht** — industrieel fort (eindbaas!)

## Development

Open `index.html` in een browser, of deploy naar GitHub Pages via de Actions workflow.
